Widget Embed Demo

Diese Seite zeigt wie die Widgets auf jeder beliebigen HTML-Seite eingebettet werden können.
Exakt wie SaniGear: <div> + <script> — fertig.

1. Öffnungszeiten Widget

<div id="bs-hours" data-config='{"regular":[...]}'></div>
<script src="/widgets/hours.js"></script>

2. Dienstleistungen Widget

<div id="bs-services" data-config='{"services":[...]}'></div>
<script src="/widgets/services.js"></script>

3. Buchungs Widget

<div id="bs-booking" data-config='{"services":[...],"barbers":[...]}'></div>
<script src="/widgets/booking.js"></script>

Embed-Optionen

<!-- Quick Embed: Widget bringt eigene Styles mit -->
<div id="bs-hours"></div>

<!-- Custom Design: Kein CSS injiziert, du stylst selbst -->
<div id="bs-hours" data-no-css="true"></div>

<!-- Config via data-Attribut -->
<div id="bs-hours" data-config='{"title":"Unsere Zeiten"}'></div>

<!-- Config via globales Objekt -->
<script>window.__BS_CONFIG__ = { title: "Global Config" };</script>
